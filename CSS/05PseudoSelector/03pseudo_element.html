What is Pseudo element ?

Ans : A CSS pseudo-element is a keyword added to a selector that lets you style a specific part of the selected elements. For Example, Styling the first letter or line of an element, and Inserting content before or after the content of an element. All of these can be done using Pseudo Elements in CSS.

Pseudo-elements defined by a set of CSS specifications include the following:
