What is symentic elements ?

= Ans : Html webpages made with different kind of components this components called as symentic elements

Example of symentic elements listed below:-

<article> this is a symentis elements in HTML5:- Defines an article

<aside> this is a symentis elements in HTML5:- Defines content aside from the page content

<details> this is a symentis elements in HTML5:- Defines additional details that the user can view or hide

<figcaption> this is a symentis elements in HTML5:- Defines a caption for a <figure> element

<figure> this is a symentis elements in HTML5:-

<footer> this is a symentis elements in HTML5:- Defines a footer for a document or section

<header> this is a symentis elements in HTML5:- Specifies a header for a document or section

<main> this is a symentis elements in HTML5:- Specifies the main content of a document

<mark> this is a symentis elements in HTML5:- Defines marked/highlighted text

<nav> this is a symentis elements in HTML5:- Defines navigation links

<section> this is a symentis elements in HTML5:- Defines a section in a document

<summary> this is a symentis elements in HTML5:- Defines a visible heading for a <details> element

<time> this is a symentis elements in HTML5:- Defines a date/time
