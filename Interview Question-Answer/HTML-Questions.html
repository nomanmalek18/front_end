1.What is a HTML

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML is used to structure a web page and its content. For example, content could be structured within a set of paragraphs, a list of bulleted points, or using images and data tables.

3. Full form of HTTP.

The full form of HTTP is Hypertext Transfer Protocol. HTTP is an application protocol that provides guidelines for transferring data and multimedia communication on the World Wide Web.

4. Full form of FTP.

FTP stands for File Transfer Protocol. It's a standard network protocol that allows users to transfer files between computers over a TCP/IP network.

5. What is a domain?

A domain is a network of computers and devices that are controlled by one authority and have specific guidelines. A domain is labeled by its domain name, such as Webopedia or webopedia.com.

A domain name is a string of text associated with the numerical IP address of a website.

For example, the domain name "google.com" points to the IP address "216.58. 216.164". Domain names make a site more accessible to users.

6. What is the full form Of URL?

The full form of URL is Uniform Resource Locator. A URL is the address of a resource on the internet, such as a webpage or file. When used with HTTP, a URL is called a web address.

10. Is HTML case sensitive?

HTML is not a Case Sensitive Language because, during parsing, all HTML elements are converted to lowercase first. HTML5, the latest version of HTML, which is a new web standard, is also case insensitive. XHTML, an older version of HTML, was case-sensitive for lowercase letters.
