Q-2. What are tags and attributes in HTML?

HTML Tags:

Angle brackets are commonly used to enclose tags, which are used to mark the beginning of an HTML element. <h1> is an illustration of a tag.

The majority of tags require an opening <h1> and closing </h1> in order to work.

Q-3. What are void elements in HTML?

The structure of each HTML tag varies. Certain elements, referred to as void elements, just need an opening tag because they themselves do not contain any elements, however the majority of elements need an opening tag, a closing tag, and contents. In this article, we'll go through how to use HTML's void elements correctly.

A void element can have characteristics but cannot contain any content. Void elements must not have a closing tag because they automatically close.

In HTML5, the following elements are void:

  • area
  • base
  • br
  • col
  • embed
  • hr
  • img
  • input
  • keygen
  • link
  • meta
  • param
  • source
  • track
  • wbr
