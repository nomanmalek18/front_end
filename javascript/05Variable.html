Variables

ES6 introduced two important new JavaScript keywords: let and const.

var declarations are globally scoped or function scoped while let and const are block scoped.

Var

the var statement declares a function-scoped or globally-scoped variable, optionally intializing it is a value.

Variables:- var

var abc = "hello";
console.log(abc);

Let

The Let statement declares a block-scoped local variable, optionally intializing it is a value.

Variables:- let

let abc = "hello";
console.log(abc);

Const

constanta are block-scoped, much like variable is declared using the let keyword. the value of a constant can't be changed through reassingment, ans it can't be redeclared

Variables:- const

const abc = "hello";
console.log(abc);
