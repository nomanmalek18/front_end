Document Object Model Selectors

DOM Id Selectors:-

<p id="para1">This is my first paragraph.</p>
let paragraph1 = document.getElementById("para1"); console.log(paragraph1);

DOM Class Selectors:-

<p class="para1">This is my first paragraph.</p>
let paragraph1 = document.getElementByClassName("para1"); console.log(paragraph1);

DOM Elements Selectors

<p class="para1">This is my first paragraph.</p>
let paragraph1 = document.getElementByName("para1"); console.log(paragraph1);

DOM Elements Selectors With Name

<p>This is my first paragraph.</p>
let paragraph1 = document.getElementByTagName(p); console.log(paragraph1);

DOM Elements by Query Selector

<p id="para1">This is my first paragraph.</p>
let paragraph1 = document.querySelectorAll("#para1); console.log(paragraph1);

DOM Elements by Query All Selector

<p class="para1">This is my first paragraph.</p> <p class="para1">This is my first paragraph.</p>
let paragraph1 = document.querySelectorAll("p.para1); console.log(paragraph1);
