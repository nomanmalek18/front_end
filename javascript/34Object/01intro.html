Object

The Object type represents one of JavaScript's data types. It is used to store various keyed collections and more complex entities. Objects can be created using the Object() constructor or the object initializer, literal syntax.

JavaScript object is a non-primitive data-type that allows you to store multiple collections of data.

Creating Objects in JavaScript

There are 3 ways to create objects.

  • object literal
  • creating instance of Object directly (using new keyword)
  • using an object constructor (using new keyword)

1: JavaScript object literal

The syntax of creating object using object literal is given below:

let UserDetailsObject = {
FName: "Noman",
LName: "Malek",
MName: "Aashik",
Gender: "Male",
Location: "Ahmedabad",
Mobile: "9879879878",
Rank: 1,
Status: true
}

console.log(UserDetailsObject);

output:

{ FName: "Noman", LName: "Malek", MName: "Aashik", Gender: "Male", Location: "Ahmedabad", Mobile: "9879879878", Rank: 1, Status: true }

creating instance of Object directly (using new keyword)

The syntax of creating object directly is given below:

let userObject = new Object();
userObject.FName = "Noman";
userObject.age = 22;
userObject.Gender = "Male",
userObject.Location = "Ahmedabad"

console.log(userObject);

output:

{FName: 'Noman', age: 22, Gender: 'Male', Location: 'Ahmedabad'}

using an object constructor (using new keyword)

The example of defining method in object is given below.

function UserDetails(id,name,salary){
this.id=id;
this.name=name;
this.salary=salary;
}
UserDetailsData=new UserDetails(1820,"Noman",30000);

console.log(UserDetailsData);

output:

UserDetails {id: 1820, name: 'Noman', salary: 30000}
